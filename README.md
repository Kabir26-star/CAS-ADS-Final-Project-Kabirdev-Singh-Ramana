# CAS-ADS-Final-Project-Kabirdev-Singh-Ramana
<h1>
  Coffee Bean Moisture Prediction During Roasting
</h1>

<h1>
  Project Overview
</h1>

<p>
  Coffee bean roasting is a critical process that significantly influences the quality, flavor, and physical properties of coffee beans. During roasting, moisture content changes dynamically as heat, environmental conditions, and processing parameters interact throughout the roasting cycle.

Understanding and accurately predicting moisture behavior is important for optimizing roasting consistency, improving product quality, and enhancing process control in coffee production.

In this project, data collected from a coffee roasting machine include environmental variables, processing conditions, and roasting parameters. These variables are analyzed to identify the factors affecting moisture content during roasting. Exploratory Data Analysis (EDA) and supervised machine learning techniques are applied to model moisture dynamics and evaluate predictive performance.
</p>

<h1>
  Objectives
</h1>
<ul>
  <li>Analyze factors influencing coffee bean moisture content during roasting.</li>
  <li>Perform exploratory data analysis (EDA).</li>
  <li>Develop machine learning models to predict moisture content.</li>
  <li>Compare model performance using evaluation metrics.</li>
  <li>Identify key variables affecting moisture behavior.</li>
</ul>
